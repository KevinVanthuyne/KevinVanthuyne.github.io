De rustigste Wekker

Met dit project is mijn passie ontstaan voor elektronica, Raspberry Pi's en Arduino's. Het was het allereeste project dat ik maakte dankzij een Raspberry die ik cadeau kreeg. Hierdoor ging ik aan het denken over welk aspect van mijn leven ik kon verbeteren. Ik concludeerde vrij snel dat opstaan vaak erg bruusk was met een liedje of alarm. Dit wilde ik verbeteren.

De oplossing die ik bedacht om zo rustig mogelijk op te staan was een wekkerradio die aanvankelijk af gaat met één willekeurig natuurgeluid. Dit ene geluid begint heel stil en wordt stilaan luider en luider. Na verloop van tijd worden nieuwe willekeurige natuurgeluiden toegevoegd tot er uiteindelijk maximaal 5 geluiden tegelijk afspelen. De wekker bevat een bibliotheek met verschillende geluiden, ingedeeld in categorieën zoals water, dieren en wind. Hierdoor speelt er telkens maar één geluid per categorie af.

De wekker kan ingesteld worden via een webinterface die gebouwd werd met PHP. Het afspelen van de geluiden en de andere logica is in Python geschreven. Aangezien dit mijn eerste ervaring was met de Raspberry en Python vind ik de code wat warrig. In de toekomst zou ik deze graag eens herwerken, liefst voor een Arduino of ESP8266 aangezien een Raspberry nogal overkill is voor een wekker. Alle code is te vinden in de Github repository van de "Python Alarm Clock"
